Guide d'utilisation du fichier Excel avec projet VBA

1) Debloquer le fichier pour autoriser les macros
- Faites un clic droit sur le fichier Excel (.xlsm), puis cliquez sur Proprietes.
- Dans l'onglet General, cochez "Debloquer" (si l'option apparait), puis cliquez sur Appliquer / OK.
- Ouvrez ensuite le fichier Excel.

2) Verifier que la feuille config est visible
- Si la feuille "config" n'est pas visible :
- Faites un clic droit sur un onglet de feuille.
- Choisissez "Afficher...".
- Selectionnez la feuille "config", puis validez.

3) Definir le Pole dans la feuille config
- Allez dans la feuille "config".
- Modifiez la colonne Pole avec le nom souhaite.
- Enregistrez puis fermez le fichier.
- Renommez le fichier en ajoutant le nom du Pole :
  de : Suivi_Pole_.xlsm
  vers : Suivi_Pole_NomDuPole.xlsm

4) Ajouter les fonctions du Pole
- Dans la feuille "config", renseignez les fonctions souhaitees dans la colonne C.
- Ajoutez autant de lignes que necessaire pour votre Pole.

5) Lors de "Mise a jour" ou "Archive"
- Si une fenetre de selection de fichier/dossier s'affiche, choisissez le dossier contenant votre fichier :
  Suivi_Pole_.xlsm (ou sa version renommee avec le nom du Pole).

Conseil
- Conservez une copie de sauvegarde avant les operations importantes (mise a jour, archive, collecte).
